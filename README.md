# ADA - Formacao 02 - Modulo 2

Repositório com os projetos didáticos desenvolvidos ao longo das aulas do Modulo 2 da formação da ADA.

O objetivo desta solution é concentrar exemplos práticos de coleções, iteração, LINQ, concorrência paralelismo e assincronismo em aplicações de console com C#.

## Estrutura

- `02_List_Dictionary`
  - Exercícios introdutórios com `List` e `Dictionary`.
- `03_IEnumerable_ICollection`
  - Exemplos com `IEnumerable`, `ICollection` e operações de consulta.
- `04_Cafe_Tech`
  - Projeto final em console baseado no cenário "Cafe Tech", reunindo:
    - `Dictionary<int, string>` para cardápio
    - `List<decimal>` para carrinho
    - uso de LINQ para soma e contagem
    - fluxo assíncrono com `async`/`await`

## Requisitos

- .NET SDK 9.0

## Objetivos de estudo

- Manipular coleções com C#
- Trabalhar com `List`, `Dictionary`, `IEnumerable` e `ICollection`
- Aplicar operações com LINQ
- Organizar pequenas aplicações em serviços e fluxo de menu
- Entender o comportamento de métodos assíncronos com `Task`, `async` e `await`

## Observações

- O projeto `04_Cafe_Tech` representa a consolidação dos conceitos trabalhados no módulo.
